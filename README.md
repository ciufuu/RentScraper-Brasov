# Scraper_Chirii
Scopul acestui script este de a extrage date de pe mai multe site-uri ce prezinta anunturi de chirii ci, in functie de dorinta utilizatorului sa filtreze rezultatele.
Momentan programul actual extrage date doar de pe olx si este destul de limitat in ceea ce priveste filtrarea datelor. 

*--- Cu timpul vor fi adaugate mai multe site-uri de pe care se vor extrage date si vor fi implementate noi functii in program! ---*
